**农历**（又称阴历、夏历）是中国古代以月亮的运行周期（即月相变化）为基础的一种历法。它结合了太阳的运动和月亮的盈亏变化，在农业、文化、宗教等方面有深远的影响

# 月份的定义
农历的一个月以月亮的运行周期为基础，一个月从新月开始，经过月亮盈亏变化，到下一个新月时结束。每个月的初一是月亮消失的日子，即新月，十五前后是满月。

+ **大月**：每个月可能是29天或30天，称为小月（29天）或大月（30天）。
+ **月相周期**：平均约为29.53天，因此农历的月份要么为29天，要么为30天。

# 年份的定义
农历的年份基于太阳的回归年（365.2422天），但农历主要是基于月亮的周期（阴历），一个阴历年大约有354天，短于太阳年。这种不匹配通过置闰月的方式来调节，使得农历与四季保持同步。

# 闰月
为了弥补阴历年（354天）和太阳回归年（365.2422天）之间的差距，农历采用了**置闰法**。即每隔2到3年增加一个闰月，使得农历的月份和四季保持一致。

+ **置闰原则**：一个阴阳历年通常有12个月（约354天），但为了与365天的阳历年保持同步，农历在19年内会插入7个闰月，使农历年的平均长度接近阳历年。
+ **闰月**：闰月通常出现在一年中的某个固定月份之后，这个闰月的存在使得该年的农历有13个月。

# Q&A
> Q：闰月怎么计算
>

> A：**闰月**的出现是为了使农历的年周期（阴历）与太阳年周期（阳历）保持一致。农历是基于月亮的朔望周期，一个朔望月大约为29.53天，因此农历一年12个月的总天数大约是354天，比阳历（365或366天）少了大约11天。为了补偿这个差距，农历采用了**闰月**的方式，每隔几年加一个闰月，以使农历年与阳历年接近。
>
> **闰月的产生和规定**
>
> **闰月**的设置基于”**19年7闰法则**“或”**置闰规则**”。这源自”**19年置7闰**“的历法规定，即19个农历年中大约有7个年需要插入闰月。这种方法可以使农历与阳历大致保持同步。
>
> **农历置闰的具体原则：**
>
> 1. **平年**：一个平年（没有闰月的年）农历有12个月，农历的年天数大约是354天，比阳历少11天。
> 2. **闰年**：为了补足这11天的差距，约每两到三年需要增加一个闰月。这样，一个闰年农历有13个月，年天数大约是383天或384天。
> 3. **依据二十四节气置闰**：农历置闰的核心依据是**二十四节气**。二十四节气是基于太阳在黄道上的位置确定的，每年24个节气平均分布在阳历年中。
>     - **12个”中气”**：二十四节气分为12个”节气”和12个”中气”，每个月必须有一个”中气”。
>     - **闰月的产生**：如果某年某个月没有中气，那么这个月就被当作闰月，称为”闰×月”。闰月通常插入在没有中气的那个农历月份之后。
>

> Q：农历大小月怎么规定
>

> A：农历的**大小月**是根据月亮的运行周期来决定的，每个月大致为29.53天，因此农历的月份分为“**大月**”和“**小月**”。
>
> + **大月**：30天
> + **小月**：29天
>
> **大小月的规定方式**
>
> 1. **大月、小月交替安排**：
>
> 农历的大小月通常交替出现，即大月30天，小月29天。这种安排使得一年12个月的天数能够尽量接近一个太阳年的长度（365.2422天）。
>
> 2. **调整和误差修正**：
>
> 因为农历的一个月平均为29.53天，12个月的总天数为354.36天，这比一个回归年（365.2422天）要少约11天。因此，农历通过**置闰月**来修正这种误差。大月和小月的分布不是固定的，根据天文观测，每个月的实际长度会略有不同。
>
> **置闰月和大小月的关系**
>
> 为了让农历和季节保持同步，每隔2到3年就会增加一个**闰月**。当增加闰月时，该年会有13个月，这也会影响大月和小月的分配。
>
> + 农历一年通常有12个月，有时会有13个月（包括一个闰月）。
> + 闰月的长度通常与其对应的前一个月相同，具体是大月还是小月取决于天文计算结果。
