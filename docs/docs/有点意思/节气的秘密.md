二十四节气过完一轮是 **365.2422天**，这与地球绕太阳一周的时间，也就是一年中的天数相同。这是因为二十四节气是根据太阳在黄道上的位置划分的，每15°为一个节气。

# 节气的计算原理
+ **黄经**：地球围绕太阳运行时，太阳在黄道（天球上的太阳的运行轨迹）上的位置叫做黄经。黄经每移动15°，就会进入下一个节气。
+ **历法**：二十四节气的日期在每年的阳历中大致固定，依据的是地球绕太阳一年的时间，即大约365.24天，因此每个节气大约相隔15天左右。
    - **冬至**到**春分**（即冬天和春天的前半段），地球公转速度较快，节气之间的间隔会略短一些，大约是**14天**左右。
    - **春分**到**秋分**（即春天、夏天、秋天的前半段），地球公转速度较慢，节气之间的间隔会略长一些，大约是**16天**左右。

# 二十四节气的分类：
+ 二十四节气分为**节气**和**中气**两个部分。
    - **节气**：指太阳在黄道上每运行 **15°** 的位置时确定的节气，如立春、立夏、立秋、立冬等。
    - **中气**：指太阳在黄道上每运行 **30°** 的位置时确定的节气，如雨水、谷雨、小满、大暑等。
+ **节气和中气的关系：**
    - 二十四节气是根据太阳在黄道上运行的轨迹划分的。每个农历月（阴历月）应该对应一个中气，如果某个月没有中气，该月后就会加一个闰月，这就是置闰的依据。
